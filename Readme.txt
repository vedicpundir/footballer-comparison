# ⚽ Footballer Comparison

A data-driven project for comparing professional footballers based on key performance metrics.

## 📌 Overview
This project allows users to compare footballers across different stats such as goals, assists, pass accuracy, defensive actions, and more — helping visualise and analyse player performance side by side.

- Python / Jupyter Notebook
- Pandas, NumPy

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- pip

### Installation
```bash
git clone https://github.com/vedicpundir/footballer-comparison.git
cd footballer-comparison
pip install -r requirements.txt
```

### Run
```bash
python main.py
```
or open the notebook:
```bash
jupyter notebook
```
